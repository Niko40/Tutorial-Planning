
io_ME3D is a script for Blender to export meshes in me3d format.

To install and activate:
- Open blender
- Press "Ctrl + Alt + U"
- Go to "Add-ons" tab
- Click install (bottom of the window)
- Browse to io_ME3D.zip and double click on it
- See the description of the Add-on and activate it if not already active
- Save User Settings if you want to keep it active

To export:
- Do your model and unwrap the UVs
- File -> Export -> Grafgear ME3D (.me3d)
- Select option for "Up Axis" and "UV Origin" on the left side
- Browse to location where you want to save, name it and click "Export ME3D"
